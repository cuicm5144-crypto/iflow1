# 骂我 - 毒鸡汤生成器 🎯

一个温柔又扎心的毒鸡汤生成器，用幽默解构压力，提供情绪出口+社交梗。

## 🌟 功能特色

- **智能生成**: 输入任何话题（老板、周一、房贷等），AI生成个性化毒鸡汤
- **文字图片**: 一键生成精美的文字图片，适合社交媒体分享
- **社交分享**: 支持分享到微博、小红书等平台
- **美观界面**: 现代化UI设计，霓虹风格视觉效果

## 🚀 快速开始

### 方法一：双击打开
直接双击 `toxic-soup-app.html` 文件，在浏览器中打开即可使用。

### 方法二：本地服务器
```bash
# 启动应用服务器
./start-app.sh

# 或手动启动
python3 -m http.server 8000
# 然后访问: http://localhost:8000/toxic-soup-app.html
```

## 📱 使用方法

1. **输入话题**: 在输入框中输入你想要的话题，如"老板"、"周一"、"房贷"等
2. **生成毒鸡汤**: 点击"生成毒鸡汤"按钮，AI会生成温柔又扎心的毒鸡汤
3. **生成图片**: 点击"生成文字图片"按钮，创建精美的分享图片
4. **分享**: 下载图片或直接分享到社交媒体

## 🎨 毒鸡汤示例

输入 "老板" 可能生成：
- "老板说你很重要，但你的KPI不这么认为。"
- "生活就像画大饼，看着很美好，实际上全是套路。"

输入 "房贷" 可能生成：
- "房贷虐我千万遍，我待房贷如初恋——除非给钱。"
- "上帝为你关上一扇门，就会为你打开一扇窗——比如房贷这扇门。"

## 🛠️ 技术特点

- **纯Web技术**: HTML5 + CSS3 + JavaScript，无需安装
- **Canvas绘图**: 动态生成精美的文字图片
- **响应式设计**: 支持手机、平板、桌面设备
- **离线可用**: 无需网络连接即可使用

## 🎯 设计理念

"用幽默解构压力，情绪出口+社交梗"

现代人生活压力大，这个应用希望通过幽默的毒鸡汤让用户在笑声中释放压力，同时提供精美的文字图片供社交分享，形成正面的情绪循环。

## 🔧 Android版本说明

由于开发环境限制，当前提供完整的Web版本作为MVP（最小可行产品）。Android版本的核心代码已准备就绪，包含：
- 完整的毒鸡汤生成逻辑
- 现代化的Material Design UI
- Canvas图片生成功能
- 分享功能集成

如需编译Android APK，需要：
1. 安装Java 8+ 和 Android SDK
2. 配置Gradle构建环境
3. 运行 `./gradlew assembleRelease`
